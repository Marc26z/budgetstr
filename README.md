# Budgetstr

![Budgetstr](https://raw.githubusercontent.com/Marc26z/budgetstr/07b81e65cbe1c2fab15743e2bc40fa74760b45fb/public/logo.jpg)

**Stay under budgetstr, stack lightning addresses**.\*

Introducing budgetstr, a vibe coded end-to-end encrypted budgeting app. It is encrypted using [NIP-44](https://github.com/paulmillr/nip44). The idea is to share a household family budget with npubs. Don't worry. Nobody needs to know what an npub is and we don't expect your spouse to practice safe nsecbunker stuff.
